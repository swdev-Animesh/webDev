Table html Element

<table> <tr><td></td></tr> </table>

table -> table row -> table data. Each tr means additional row and each td means column in that row.

Table can be used to put layout . And also for nested tables. Table can have any tag inside the td . table inside table , div inside table , images ...everything can get inside table , it can help in layouting .

Table element can have

  • table head -> thead
  • table body -> tbody
  • table fooer -> tfoot
  • table can have broder in each cell -> border="1" attribute set to 0 or 1 if we dont or do want broder
  • Table head in bold can be given by th instead of regular td
  • rows and columns -> rowspan="" colspan="" -> attributes ot td
  • It can have a caption right after starting table tag that tells us about the table .
  • Border spacing and cell padding for styling

Horizontal Rule

<hr>

This creates a 3d seperation between sections . it is block element . we can put attributes such as

  • height=""
  • width=""
  • size=""
etc .. all the attributes of a div can be put here as hr is just some div with very less height.

For styling we have to style its border , and if the hr is thick then we can apply background color too.

  • border-style-type
  • box-shadow
  • background-color
  • border-color
  • border-style-type-top
etc..
